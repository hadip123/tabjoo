# TabJoo

A Firefox addon to quickly search and switch between open tabs using `Ctrl+Shift+Space`.

Type to search — results are scored by title matches, URL matches, domain, fuzzy character matching, and recency. Navigate with `^N`/`^P` or arrow keys, hit `Enter` to switch.

**Note:** This addon was generated with AI assistance.

## Usage

- Press `Ctrl+Shift+Space` to open the search popup
- Type part of a tab title or URL
- Use `^N`/`^P` (or arrow keys) to select a result
- Press `Enter` to switch to that tab
- Press `Esc` to close
